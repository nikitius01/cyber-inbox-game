# Security Notes

- Правильный ответ не отправляется на фронтенд до ответа игрока.
- RAW HTML письма показывается как текст, без `dangerouslySetInnerHTML`.
- Ссылки отображаются как данные для анализа, не как прямые переходы.
- Вложения являются метаданными, не реальными исполняемыми файлами.
- OpenAI API key используется только на бэкенде.
- Все JSON-запросы ограничены по размеру.
- Добавлены CORS, rate limit и базовые security headers.
